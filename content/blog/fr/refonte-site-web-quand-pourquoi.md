Un site web, c'est comme une maison : avec le temps, les murs se fissurent, les normes changent, les besoins évoluent. **Quand faut-il refondre** ? Voici les signaux qui ne trompent pas.

## Les 7 signaux pour refondre

1. **Votre site a plus de 4 ans** sans refonte visuelle majeure.
2. **Il n'est pas responsive** ou mal optimisé mobile.
3. **Le PageSpeed < 50** sur mobile (gros signal négatif pour Google).
4. **Votre taux de rebond > 70 %** sur les pages d'entrée.
5. **Votre techno est obsolète** (PHP 7.x, jQuery majoritaire, CMS abandonné).
6. **Vos métiers ont évolué** et le site ne reflète plus votre offre.
7. **Vos concurrents ont tous pris 2 générations d'avance visuelle.**

## Les 4 types de refonte

- **Cosmétique** : nouveau thème, charte graphique. 3-8 k€, 4-6 semaines.
- **Structurelle** : nouvelle arborescence, nouveaux contenus, même CMS. 8-20 k€, 2-4 mois.
- **Technologique** : changement de CMS (ex: vieux WordPress → Next.js). 20-50 k€, 3-6 mois.
- **Stratégique** : refonte totale avec repositionnement marketing. 30-80 k€, 4-9 mois.

## Comment ne PAS perdre son SEO

La refonte est le **moment n°1** où les sites perdent 30-80 % de leur trafic SEO. Checklist impérative :

- Cartographier toutes les URLs existantes qui reçoivent du trafic (Search Console)
- Créer un **plan de redirections 301** exhaustif (ancienne URL → nouvelle)
- Garder au minimum les mêmes H1, balises title et meta descriptions pour les pages qui marchent
- Prévenir Google via Search Console (soumettre le nouveau sitemap immédiatement)
- Monitorer pendant 90 jours : trafic, erreurs 404, pages désindexées

## Les étapes d'une refonte réussie

1. **Audit** : UX, SEO, technique, contenu (2-3 semaines)
2. **Stratégie & arborescence** : validation des parcours utilisateurs
3. **Wireframes + design** : maquettes cliquables Figma
4. **Développement** : intégration + CMS + optimisations
5. **Recette + tests** : cross-browser, mobile, performance, accessibilité
6. **Mise en ligne avec plan de redirections**
7. **Monitoring SEO + UX 90 jours**

## Les pièges à éviter

- **Couper le site ancien avant d'avoir vérifié toutes les redirections.** Vous perdez votre référencement.
- **Sous-estimer la production de contenu.** C'est souvent 40 % du temps global.
- **Vouloir tout refaire en 6 semaines.** Une refonte bâclée coûte plus cher qu'une refonte reportée.
- **Négliger la phase de recette.** Un bug sur le formulaire de contact = 0 lead pendant 2 semaines.

## En conclusion

Une refonte, ce n'est pas un caprice de designer. C'est un **investissement business** qui doit améliorer le taux de conversion, le SEO, et l'expérience utilisateur. Si ces 3 axes ne sont pas explicitement travaillés, vous faites juste du relooking coûteux.
